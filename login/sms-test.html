接口文档

POST /api/sms/send
发送短信验证码

请求参数:
{
    "phone": "string (必填, 11位手机号)",
    "codeLength": "number (可选, 默认6)"
}

成功响应:
{
    "success": true,
    "message": "验证码发送成功",
    "expireMinutes": 5
}

失败响应:
{
    "success": false,
    "error": "INVALID_PHONE | PHONE_RATE_LIMITED | PHONE_DAILY_LIMIT",
    "message": "错误描述"
}
POST /api/sms/verify
验证短信验证码

请求参数:
{
    "phone": "string (必填, 11位手机号)",
    "code": "string (必填, 验证码)"
}

成功响应:
{
    "success": true,
    "message": "验证成功",
    "verifiedAt": "timestamp"
}

失败响应:
{
    "success": false,
    "error": "CODE_NOT_FOUND | CODE_EXPIRED | CODE_MISMATCH | CODE_MAX_ATTEMPTS",
    "message": "错误描述"
}
GET /api/sms/status/{phone}
查询验证码状态

成功响应:
{
    "sent": true|false,
    "status": "not_sent | pending | verified | expired",
    "remainingAttempts": number,
    "remainingSeconds": number
}
